## 2. 색상 시스템

### 핵심 색상 (CSS 변수 기반)

| 변수명 | 역할 | 추정 계열 |
|--------|------|-----------|
| `--main-b` | 브랜드 Primary | 블루/네이비 계열 |
| White `#FFFFFF` | 배경, 텍스트 반전 | — |
| Dark (Near Black) | 본문, 네비게이션 | `#1A1A1A` 추정 |

### 색상 사용 원칙

- **배경**: 화이트 기반, 섹션별 그레이(#F5F5F5) 분리
- **텍스트**: 다크 컬러 단일 사용 / 강조 시 브랜드 컬러
- **CTA 버튼**: 브랜드 Primary 배경 + 흰 텍스트
- **배지/태그**: 이벤트 상태(pre-sale, on-sale, last-day)에 따라 동적 컬러 분기

### VIP 티어 컬러

VIP 구독 혜택 강조를 위해 골드/허니 계열 포인트 컬러 별도 사용

## 3. 타이포그래피

### 폰트 스택

```css
font-family: 'Apple SD Gothic Neo', 'Noto Sans KR', 'Malgun Gothic', sans-serif;
/* Cafe24 플랫폼 기반, 한글 최적화 폰트 우선 순위 */
```

### 사이즈 스케일

| 용도 | 크기 | 굵기 |
|------|------|------|
| 히어로 헤딩 | 48px ~ 60px | 900 |
| 섹션 타이틀 | 32px | 900 |
| 카드 상품명 | 16px ~ 18px | 600 |
| 본문 | 14px ~ 15px | 400 |
| 배지/태그 | 11px ~ 12px | 600 |
| 가격 | 16px ~ 20px | 700 |

### 타이포 원칙

- 섹션 제목에 **영문 서브타이틀** 병기 (예: "BEST SELLER", "NEW ARRIVAL")
- 줄간격(line-height) 넉넉하게 1.6 ~ 1.8 유지
- 강조 키워드는 별도 컬러 또는 굵기 900으로 처리

## 4. 레이아웃 & 그리드

### 컨테이너

```
최대 너비: 1200px (추정)
좌우 패딩: 24px (모바일), 48px (데스크톱)
```

### 그리드 패턴

| 섹션 | 컬럼 구성 |
|------|-----------|
| 히어로 배너 | 풀 와이드 (100vw) |
| 상품 그리드 | 4열 (데스크톱) → 2열 (태블릿) → 1열 (모바일) |
| 카테고리 탭 | 수평 스크롤 리스트 |
| 기능 소개 | 2열 또는 교차 배치 (이미지 + 텍스트) |
| 푸터 | 3~4열 링크 그룹 |

### 반응형 전략

- **Mobile-first** 기반 (Cafe24 플랫폼 표준)
- Swiper.js 캐러셀로 모바일에서 그리드 → 슬라이더 전환
- 스티키 헤더 (스크롤 시 배경 불투명 처리)

## 5. 섹션 구성 (홈페이지 순서)

```
① 상단 이벤트 배너 캐러셀   ← 날짜 기반 동적 전환
② 글로벌 네비게이션 헤더
③ 히어로 섹션               ← "THE REASON WE EXIST"
④ 디자인 어워드 배지 섹션   ← Red Dot, German Design Award
⑤ VIP 정기배송 혜택 캐러셀
⑥ 베스트셀러 상품 그리드
⑦ 신상품(New Arrival) 캐러셀
⑧ Daily Care 영양 라인 소개
⑨ PETHROOM Friends 앱 섹션
⑩ IoT 스마트 디바이스 특징
⑪ CSR / 사회적 책임 콘텐츠
⑫ 푸터
```

**섹션 설계 포인트**
- 상업적 콘텐츠(상품)와 브랜드 스토리(어워드·CSR) 교차 배치로 신뢰 구축
- VIP 구독 혜택이 전체 페이지에 걸쳐 반복 노출

## 6. 컴포넌트 패턴

### 6-1. 네비게이션

```
[로고]  [ALL] [NEW] [BEST] [FOR DOG] [FOR CAT] [COLLABO]  [검색] [장바구니]
```
- 카테고리 hover 시 드롭다운
- 스크롤 시 배경 solid 전환 (투명 → 흰색)

### 6-2. 상품 카드

```
┌─────────────────────┐
│   상품 이미지        │  ← 호버 시 대체 이미지 or 확대
│   [BEST] 배지       │
├─────────────────────┤
│ 상품명               │  font-weight: 600
│ ★★★★☆ (4.8) 리뷰수 │
│ 정가 ~~12,000원~~   │
│ 이벤트가 9,600원    │
│ VIP가 6,240원       │  ← 3단 가격 구조
│         [🛒 담기]   │
└─────────────────────┘
```

**3단 가격 구조**가 핵심 차별화 요소:
1. 정가 (취소선)
2. 이벤트가
3. VIP가 (35% 추가 할인)

### 6-3. CTA 버튼

| 타입 | 스타일 |
|------|--------|
| Primary | 브랜드 컬러 배경 + 흰 텍스트 + 화살표 아이콘 |
| Secondary | 테두리(outline) + 브랜드 컬러 텍스트 |
| 장바구니 | 아이콘 오버레이 (카드 호버 시 노출) |

### 6-4. 배너 (이벤트 상태 분기)

```javascript
// 날짜 기반 4가지 상태로 배너 동적 교체
.event-pre      // 사전 예약 기간
.event-main     // 이벤트 기간
.event-lastday  // 마지막 날
.event-normal   // 일반 기간
```

### 6-5. 캐러셀 / 슬라이더

- **라이브러리**: Swiper.js
- **자동재생**: 5초 간격
- **방향**: 수평 (상품), 수직 (VIP 혜택 텍스트 슬라이더)
- **페이지네이션**: 점(dot) 또는 숫자

### 6-6. 배지 / 태그

| 배지 | 색상 | 용도 |
|------|------|------|
| BEST | 브랜드 컬러 | 인기 상품 |
| NEW | 그린 계열 | 신규 입고 |
| VIP | 골드 계열 | 구독 전용 혜택 |
| 할인율 | 레드 계열 | 가격 강조 |

## 7. 인터랙션 패턴

| 요소 | 인터랙션 |
|------|----------|
| 상품 카드 | 호버 시 이미지 전환 + 장바구니 버튼 노출 |
| 헤더 | 스크롤 시 배경 불투명 전환 |
| 배너 | 날짜 조건에 따라 서버에서 클래스 동적 적용 |
| 캐러셀 | 자동재생 + 스와이프 제스처 |
| 이미지 보호 | 우클릭 차단, 드래그 방지, F12/PrintScreen 방지 |

## 8. 브랜드 신뢰 요소 배치

```
어워드 배지       → 섹션 전용으로 상단 노출
고객 리뷰         → 별점 + 텍스트 + 사진 조합
VIP 혜택 수치     → "35% 추가 할인" 구체적 숫자 강조
CSR 콘텐츠        → 하단 섹션에서 브랜드 철학 전달
앱 연동 (Friends) → 락인(lock-in) 강화
```
