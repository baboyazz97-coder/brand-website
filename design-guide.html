06

인터랙션 패턴

사용자 동작에 반응하는 UI 인터랙션 정의입니다.

요소 트리거 반응 CSS / 방법
상품 카드 hover 위로 6px 이동 + 그림자 심화 translateY(-6px) + shadow-lg
카테고리 카드 hover 위로 4px + 테두리 Primary 전환 translateY(-4px) + border-color: primary
스티키 헤더 스크롤 배경 투명 → 반투명 흰색 + 블러 rgba(255,255,255,0.95) + backdrop-filter
Primary 버튼 hover 배경 어두워짐 + 1px 상승 primary-dark + translateY(-1px)
Outline 버튼 hover 배경 채워짐 (Outline → Filled) background: primary; color: #fff
장바구니 버튼 hover Light 배경 → Primary 배경 primary-light → primary
히어로 이미지 상시 부유 애니메이션 (float) @keyframes float 3s ease-in-out infinite
배너 띠 상시 좌→우 무한 스크롤 @keyframes scroll 20s linear infinite
찜 버튼 hover / active 회색 → 레드 (활성 시 채우기) color: #E53935
이벤트 배너 날짜 조건 CSS 클래스 동적 교체 JS Date 분기 → DOM 클래스 주입
